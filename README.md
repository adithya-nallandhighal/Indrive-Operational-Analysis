# Indrive-Operational-Analysis
# 🚖 inDrive Operational Performance Analytics Dashboard

An End-to-End Business Intelligence project that analyzes ride-booking operations to uncover insights into ride completion, cancellations, customer behavior, driver performance, operational efficiency, and revenue trends using *Excel, MySQL, and Power BI*.

---
## 📌 Project Overview
This End-to-End Business Intelligence project that analyzes ride-booking operations to uncover trends in bookings, cancellations, customer satisfaction, driver performance, and revenue. The dataset was cleaned and transformed using Excel, analyzed with MySQL, and visualized through interactive Power BI dashboards. The project tracks key operational KPIs, identifies performance bottlenecks, and provides actionable insights to support data-driven business decisions. It demonstrates a complete analytics workflow from raw data to executive-ready dashboards.

---

## 🛠️ Tech Stack

- 📊 Microsoft Excel – Data Cleaning & Preparation
- 🗄️ MySQL – Data Analysis & Querying
- 📈 Power BI – Dashboard Development & Visualization

---

## 🎯 Business Objectives

- Improve Ride Completion Rate
- Reduce Customer & Driver Cancellations
- Analyze Revenue & Revenue Leakage
- Monitor Customer & Driver Ratings
- Identify Peak Booking Trends
- Discover High-Demand Pickup & Drop Locations
- Evaluate Vehicle Performance
- Support Data-Driven Business Decisions

---

## 📊 Key Performance Indicators (KPIs)

- Total Bookings
- Successful Bookings
- Ride Completion Rate
- Cancellation Rate
- Total Revenue
- Revenue Lost
- Average Booking Value
- Average Ride Distance
